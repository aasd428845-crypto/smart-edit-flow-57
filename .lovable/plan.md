
# مونتاجي الذكي - تطبيق مونتاج بالذكاء الاصطناعي

## الصفحة الرئيسية
- واجهة عربية (RTL) بتصميم هادئ وألوان مريحة
- عنوان "مونتاجي الذكي - للإستخدام الشخصي"
- منطقة سحب وإفلات لرفع فيديو (mp4, mov) مع أيقونة واضحة
- زر "رفع إلى Vimeo" يظهر بعد اختيار الفيديو
- بعد الرفع الناجح: مربع نص كبير لكتابة أمر المونتاج + زر "تنفيذ المونتاج"
- شريط تقدم يعرض المراحل: جاري التفريغ → جاري القص → جاري الترجمة → انتهى
- رابط تحميل الفيديو النهائي عند الانتهاء

## صفحة الإعدادات
- صفحة منفصلة لإدخال Vimeo Access Token
- حفظ التوكن في localStorage (بدون تسجيل دخول)
- رابط للإعدادات من الصفحة الرئيسية

## قاعدة البيانات (Supabase)
- جدول `projects` يحتوي على: project_id, video_url, user_command, status, output_url, created_at
- تتبع الحالة في الوقت الحقيقي عبر Supabase Realtime

## التكاملات
- **Vimeo API**: رفع الفيديو عبر Edge Function تستقبل التوكن وتتعامل مع Vimeo API
- **الخادم المحلي**: إرسال POST إلى `http://localhost:8000/process` مع بيانات المشروع (video_url, command, project_id)

## التصميم
- اتجاه RTL كامل، خط عربي واضح
- ألوان هادئة (أزرق/رمادي) مع تدرجات لطيفة
- تصميم مُحسّن للحاسوب
